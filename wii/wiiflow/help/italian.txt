
Comandi:
- Menu principale (coverflow):
-- Su / giù = Precedente / prossimo gioco (verticale)
-- Sinistra / destra = Precedente / prossimo gioco (orizzontale)
-- Meno / più = Scorri velocemente tra i giochi
-- A = Seleziona gioco
-- B = Mostra Source Menu (se installato)
-- B+A = Avvia gioco immediatamente
-- B+home = Ricarica wiiflow
-- Home = Mostra menu principale / Esci da WiiFlow
-- Home (sourceflow/menu)  =  Configura source menu
-- Home (nella vista homebrew) = Congigura modalità homebrew
-- 1 / 2 = Precedente / prossima disposizione coverflow
-- B+sinistra / B+destra = Cambia canzone
-- B+su / B+giù = Gioco Precedente / successivo nella lista(ordine alfabetico predefinito)
-- B+frecce sullo schermo = Ricerca alfabetica
-- B+meno = Avvia o seleziona un gioco casuale del coverflow
-- B+più = Ordine giochi (alfabetico, wifi, giocatori, numero partite, etc...)
-- Z / ZR = Prendi Screenshot
-- A sulle icone a freccia  =  Scorrimento veloce giochi
-- B sulle icone a freccia = Gioco Precedente / successivo nella lista(ordine alfabetico predefinito)
-- A sull'icona a stella = Preferiti
-- A sull'icona Categorie  =  Seleziona le categorie da usare
-- A sull'icona a ingranaggio = Menu impostazioni principali
-- A sull'icona USB(Wii), GC, Canali, Plugin, o Homebrew = Vai a questa modalità
-- A sull'icona Casa = Mostra menu principale / Esci da WiiFlow
-- A sull'icona Disco (se presente un disco) = Avvia il disco

- Gioco:
-- A sulla cover = Mostra lato posteriore
-- A fuori dallo schermo = Avvia gioco
-- A sull'icona Categorie = Imposta le categorie per il gioco corrente
-- B = Ritorna al coverflow
-- Su / giù = Precedente / prossimo gioco (verticale)
-- Sinistra / destra = Precedente / prossimo gioco (orizzontale)
-- Più = Informazioni sul gioco
-- Meno = Riproduci video del gioco (se disponibie)
-- Destra (in informazioni sul gioco) = Descrizione del gioco
-- Sinistra (in descrizione del gioco) = Informazioni sul gioco
-- A sull'icona a stella = Imposta come favorito (giallo = favorito)
-- A sull'icona a mano = Blocco parentale (rosso = bloccato)
-- A sull'icona a ingranaggio = Impostazioni gioco
-- A sull'icona a X = Elimina gioco

- Vista posteriore copertine:
-- B  =  Reimposta visuale predefinita
-- Su / Giù / Sinistra / Destra  =  Sposta visuale
-- Plus  =  Ingrandisci visuale
-- Minus = Riduci visuale

- Menu Impostazioni (tutti):
-- Meno / più = Precedente / prossima pagina
-- Sinistra / destra = Precedente / prossima pagina
-- Su / Giù = Pulsante precedente / successivo

- Menu Modifica coverflow:
-- B+meno / B+più = Pagina precedente / successiva
-- B = Modifiche più rapide (B+A invece di A per cliccare un pulsante)
-- B+1 = Copia l'intero coverflow
-- B+2 = Incolla coverflow

CONSIGLI per avviare giochi che crashano:
Se il gioco si blocca mostrando una schermata nera all'avvio, controlla se è un gioco PAL.
(Premi il pulsante "+" sulla schermata del gioco.) Se è PAL, imposta la modalità video a "PAL 50Hz" nelle impostazioni del gioco.
Puoi anche usare un cavo component e settare la wii a 480p per risolvere il problema.
