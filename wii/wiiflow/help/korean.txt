컨트롤 :
- 메인 메뉴 (커버플로우) :
- 위 / 아래 = 이전 / 다음 게임 (세로)
- 왼쪽 / 오른쪽 = 이전 / 다음 게임 (가로)
- "-" / "+" = 게임의 빠른 검색
- A = 게임 선택
- B + A = 즉시 게임 시작
- B + Home = 위플로우 재로드
- Home = 홈 메뉴 열기 / 위플로우 종료
- 1/2 = 이전 / 다음 커버플로우 모드
- B + 왼쪽 / B + 오른쪽 = 노래 변경
- B + 위 / B + 아래 = 알파벳순 검색
- B + 화면의 화살표 = 알파벳순 검색
- B + "-" = 파티션 변경
- B + "+" = 게임 정렬

- 별 아이콘 = 즐겨찾기
- 별 아이콘 B = 카테고리
- 기어 아이콘 = 위플로우 설정
- 기어 아이콘 B = 캐시 새로고침
- 위, 게임큐브, 채널, 플러그인 또는 홈브류 아이콘 A = 해당보기로 전환
- 위, 게임큐브, 채널, 플러그인 또는 홈브류 아이콘 B - 에뮤-낸드 활성화 / 비활성화 및 채널보기로 전환
- 집 아이콘 A = 위 메뉴로 나가기
- 집 아이콘 B (wiimote_gestos 활성화) = B를 3초 동안 누름
- 디스크 아이콘 = 게임 디스크 실행
- 물음표 아이콘 = 크레딧 및 이 도움말 표시

- 게임 :
- 상자에 뒷면 표시 = 뒷면
- 꺼짐 화면 = 게임 시작
- B = 커버플로우로 돌아가기
- 위 / 아래 = 이전 / 다음 게임 (세로)
- 왼쪽 / 오른쪽 = 이전 / 다음 게임 (가로)
- "+" = 게임 정보
--- 오른쪽 (게임 정보에서) = 게임 설명
--- 왼쪽 (게임 설명에서) = 게임 정보
- 별 아이콘에 A = 즐겨찾기 활성화 (노란색 = 즐겨찾기)
- 별 아이콘에 B 표시 = 게임 카테고리 설정
- 손에 아이콘 = 자녀 보호 (빨강색 = 잠김)
- 기어 아이콘 = 게임 설정
- X 아이콘 = 게임 삭제

- 메뉴 설정 :
- "-" / "+" = 이전 / 다음 페이지
- 왼쪽 / 오른쪽 = 이전 / 다음 페이지

- 커버플로우 설정 구성 :
- B + "-" / B + "+" = 이전 / 다음 페이지
- B = 빠른 설정 (버튼을 클릭하려면 'A' 대신 'B' + 'A')
- B + 1 = 커버플로우 복사
- B + 2 = 커버플로우 삽입

차단된 게임에 대한 팁 :
게임 멈추고 검은 화면이 나타마면 게임이 해당 지역이 맞는지 확인하세요.
(게임 설정으로 이동합니다.) 해당 지역이 아닌 경우 비디오 모드를 게임 지역으로 설정하세요.
또한 컴포넌트 케이블을 사용하여 480p 로 설정할 수 있습니다.

게임이 충동하는 경우 그냥 새로고침을 해 보세요. 계속 문제가 발생하면 게임 설정에서 USB 에뮬레이션을 "아니오"로 변경하십시오.
